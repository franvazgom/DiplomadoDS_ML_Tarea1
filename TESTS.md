# Ejercicios Básicos!

## Configuración de pruebas con **pytest**

`nota:` para todo las pruebas puedes usar esta configuración:
### Setup command (optional)
```
sudo -H pip3 install pytest
```

    ### Run command
    ```
    pytest assignments/00HelloWorld
    ```
